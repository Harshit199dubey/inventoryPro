
----------------- To Run this -------------

==== first Start Api ====

step 1 : open terminal.
step 2 : cd ./api.
step 3 : npm i.
step 4 : npm start

=== then Start parcel ==

Step 1 : open new terminal
step 2 : npm i 
step 3 : npm start



******* To Run this Web application You have to run both api and parcel **********
